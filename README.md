# Smart Calculator Pro

A professional calculator web application built using HTML, CSS and JavaScript.

## Features

- Scientific Calculator
- Voice Calculator
- History Tracking
- Export History
- Dark/Light Mode
- BMI Calculator
- Age Calculator
- EMI Calculator
- Keyboard Support
- Responsive UI
- Progressive Web App

## Technologies

- HTML5
- CSS3
- JavaScript
- Local Storage
- Web Speech API
- jsPDF

## Run

Open index.html in browser.
